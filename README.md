# Movie_Dataset_Analyser
This project performs exploratory data analysis (EDA) on a movie dataset containing information such as movie titles, genres, language, popularity, vote count, and poster information.  Here i extract useful insights from movie data using Python data analysis libraries and visualize patterns such as the most common genres, yearly movie releases.


📊 Project Objectives
--
The analysis answers several important questions from the dataset, including:

Which genre appears most frequently in the dataset?

In which year were the most movies released?

What is the distribution of movies across different genres?

Which movies have highest popularity and vote counts?

How does movie production vary by year?

📈 Key Insights & Visualizations
--
The project includes:

📊 Genre Count Analysis
--
Frequency of each movie genre

Bar graph visualization of genre distribution

📅 Year-wise Movie Release Analysis
--
Identifies the year with the highest number of movie releases

Visualizes yearly trends

⭐ Popularity Analysis
--
Distribution of movie popularity scores

🗳 Vote Count Analysis
--
Comparison of movies based on vote counts

🛠️ Technologies Used
--
Python

Pandas – Data cleaning and analysis

NumPy – Numerical computations

Matplotlib – Data visualization

Seaborn – Statistical data visualization
